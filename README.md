🚗 Penta 2026 自駕環島行控中心

這是一個為 2026 年環島自駕旅行設計的單頁互動式應用程式 (SPA)。透過視覺化儀表板，整合了行程時間軸、天氣預報、駕駛強度分析以及美食導覽。

🌟 核心功能

互動式路線地圖：使用 Plotly.js 繪製抽象環島路徑，快速掌握地理位置。

每日指揮中心：提供 Day 1 至 Day 8 的詳細時間軸、天氣提醒與注意事項。

動態數據分析：

節奏分析：視覺化每日駕駛、探索與用餐的時間佔比。

駕駛強度曲線：分析每日行車壓力，提醒駕駛者合理分配體力。

美食導覽系統：可根據城市篩選推薦餐廳，並附帶特色註解。

自駕安全提醒：針對特殊路段（如蘇花、南迴、台南市區）提供專業駕駛建議。

🛠️ 技術棧

Frontend: Tailwind CSS (Responsive Layout)

Charts: Chart.js (Doughnut & Line charts)

Geospatial Viz: Plotly.js (Abstract Map)

Data Persistence: In-memory JavaScript arrays & objects.
